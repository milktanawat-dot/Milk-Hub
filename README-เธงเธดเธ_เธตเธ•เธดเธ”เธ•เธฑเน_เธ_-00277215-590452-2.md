# Milk Hub PWA — วิธีติดตั้งใช้งาน

โฟลเดอร์นี้มี 6 ไฟล์: `index.html` (ตัวแอป), `manifest.json`, `sw.js`, และไอคอน 3 ไฟล์ — ต้องอัพโหลดขึ้นเว็บ **ครบทุกไฟล์ ไว้โฟลเดอร์เดียวกัน**

## ขั้นที่ 1 — เอาขึ้น GitHub Pages (ฟรี, ~10 นาที, ทำครั้งเดียว)

1. สมัคร/ล็อกอิน github.com
2. กด **New repository** → ตั้งชื่อ เช่น `milk-hub` → เลือก **Private ไม่ได้** สำหรับ Pages ฟรี ต้องเป็น **Public** (ไม่ต้องกังวล — ในไฟล์ไม่มีข้อมูลการเงินจริง ข้อมูลอยู่ใน localStorage เครื่องเราเท่านั้น) → Create
3. หน้า repo กด **uploading an existing file** → ลากไฟล์ทั้ง 6 ไฟล์ลงไป → Commit changes
4. ไปที่ **Settings → Pages** → Source เลือก **Deploy from a branch** → Branch เลือก `main` / root → Save
5. รอ 1-2 นาที จะได้ URL: `https://ชื่อยูสเซอร์.github.io/milk-hub/`

## ขั้นที่ 2 — ย้ายข้อมูลเดิม (ทำครั้งเดียว)

1. เปิดไฟล์ HTML ตัวเดิมในเครื่อง → ⚙️ ตั้งค่า → **ดาวน์โหลด Backup (JSON)**
2. เปิด URL ใหม่บนมือถือ/คอม → ⚙️ ตั้งค่า → **นำเข้าจากไฟล์ Backup**

## ขั้นที่ 3 — ติดตั้งเป็นแอปบนมือถือ

- **iPhone (Safari):** เปิด URL → กดปุ่มแชร์ ⬆️ → **เพิ่มลงหน้าจอโฮม**
- **Android (Chrome):** เปิด URL → เมนู ⋮ → **เพิ่มไปยังหน้าจอหลัก** (หรือจะมี banner "ติดตั้งแอป" เด้งขึ้นเอง)

จะได้ไอคอน M สีม่วงบนหน้าจอ เปิดแล้วเต็มจอเหมือนแอปจริง ใช้งาน offline ได้ (ยกเว้นราคาหุ้น/คริปโต/OCR ที่ต้องต่อเน็ต)

## ขั้นที่ 4 — เชื่อม OneDrive (ไม่บังคับ แต่แนะนำมาก)

ป้องกันข้อมูลหายถ้าเผลอล้าง browser data

1. ไปที่ **portal.azure.com** ล็อกอินด้วยบัญชี Microsoft ที่ใช้ OneDrive (ฟรี ไม่เสียเงิน)
2. ค้นหา **App registrations** → **New registration**
   - Name: `Milk Hub`
   - Supported account types: **Personal Microsoft accounts only**
   - Redirect URI: เลือก platform **Single-page application (SPA)** ใส่ `https://ชื่อยูสเซอร์.github.io/milk-hub/` (URL จากขั้นที่ 1 ต้องตรงเป๊ะรวม / ท้าย)
3. กด Register → คัดลอก **Application (client) ID** ในหน้า Overview
4. เปิด Milk Hub → ⚙️ ตั้งค่า → การ์ด **☁️ OneDrive Sync** → วาง Client ID → กด **เชื่อมต่อ OneDrive** → ล็อกอิน Microsoft
5. ติ๊ก **สำรองอัตโนมัติ** — จากนี้ทุกครั้งที่บันทึกรายการ ระบบจะอัพโหลด backup ขึ้น OneDrive ให้เอง (เก็บที่ OneDrive → Apps → Milk Hub)

**ใช้หลายเครื่อง:** เครื่องหลักเปิด auto-sync ไว้ เครื่องรองกด "กู้คืนจาก OneDrive" เพื่อดึงข้อมูลล่าสุด (ระวัง: กู้คืน = เขียนทับข้อมูลในเครื่องนั้น)

## ฟีเจอร์ใหม่ 💬 บันทึกด่วนแบบแชท

กดปุ่มม่วง 💬 มุมขวาล่าง (มีทุกหน้า) แล้ว:

- **พิมพ์เอง:** `กาแฟ 40` · `เมื่อวาน มื้อเย็น 150` · `เงินเดือน 30000` · `12/7 ตัดผม 80` · `ลงทุน dime 1,000 บาท` — ระบบแยกวันที่/ประเภท/หมวด/จำนวนเงินให้อัตโนมัติ พิมพ์หลายรายการแยกบรรทัดได้
- **แนบสลิป/ใบเสร็จ:** กด 📎 เลือกรูป ระบบ OCR อ่านยอดเงิน+วันที่ให้
- ทุกรายการจะขึ้น**การ์ดให้เช็คก่อน** แก้หมวด/จำนวนได้ แล้วค่อยกด ✓ บันทึก

## อัพเดทแอปในอนาคต

แก้ไฟล์แล้วอัพโหลดทับใน GitHub repo เดิม → เปิดแอปตอนต่อเน็ตจะได้เวอร์ชันใหม่เอง (ข้อมูลไม่หายเพราะอยู่ใน localStorage ไม่ใช่ในไฟล์)
